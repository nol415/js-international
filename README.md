# JS International Website

GitHub Pages 업로드용 최종 정리본입니다.

## 반영 내용
- 최종 컨셉 디자인 적용
- 상단 메뉴 순서: 홈 / 회사소개 / 산업분야 / 제품소개 / 문의하기
- 제품군: 스퍼터링 타겟 추가
- 상세페이지: 제품군 및 산업분야 페이지 포함
- Footer: 대표자명 제거, 주소 수정, Fax 추가

## 업로드 방법
압축을 풀고, 안쪽 파일 전체(index.html, styles.css, script.js, assets 폴더 등)를 GitHub 저장소 루트에 업로드하세요.
