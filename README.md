# JS International 홈페이지 파일

이 폴더는 바로 업로드해서 사용할 수 있는 정적 홈페이지 파일입니다.

## 파일 구성
- `index.html` : 메인 페이지
- `styles.css` : 전체 스타일
- `script.js` : 모바일 메뉴 열기/닫기
- `assets/` : 로고, 제품 이미지, 배경 이미지

## GitHub에 올리는 방법
1. 이 폴더를 압축 해제합니다.
2. GitHub 저장소 화면에서 **uploading an existing file**를 클릭합니다.
3. 압축 해제한 폴더 안의 파일과 `assets` 폴더를 모두 드래그해서 업로드합니다.
4. 아래쪽의 **Commit changes**를 눌러 저장합니다.

## 배포 방법
### 방법 1: Vercel
1. Vercel에 로그인합니다.
2. GitHub 저장소를 연결합니다.
3. 배포 버튼을 누릅니다.
4. 배포 완료 후 생성된 주소로 접속합니다.

### 방법 2: GitHub Pages
1. 저장소의 **Settings**로 이동합니다.
2. **Pages** 메뉴를 엽니다.
3. Branch를 `main` / root로 선택합니다.
4. 저장하면 공개 주소가 생성됩니다.

## 수정 포인트
- 회사 정보: `index.html` 하단 `footer` 부분
- 메인 문구: `index.html`의 `hero` 섹션
- 제품 이미지: `assets` 폴더 안 파일 교체
- 색상/간격: `styles.css`
